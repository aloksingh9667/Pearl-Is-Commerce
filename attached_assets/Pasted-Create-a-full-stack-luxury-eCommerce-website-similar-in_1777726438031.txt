Create a full-stack luxury eCommerce website similar in experience to Myntra, focused on selling jewelry such as pendants, necklaces, rings, bracelets, and accessories.

### 🧱 TECH STACK
- Frontend: Next.js (React), Tailwind CSS, Framer Motion (for animations)
- Backend: Node.js with Express
- Database: PostgreSQL
- Authentication: Google Login (OAuth) + Email/Password
- Email Service: Resend (for email verification and notifications)
- Payment Gateway: Razorpay (with receipt generation and digital signature verification)
- Storage: Cloudinary or AWS S3 for product images
- State Management: Redux Toolkit or Zustand
- API Architecture: RESTful APIs
- Deployment Ready

---

### 🎨 DESIGN REQUIREMENTS (VERY IMPORTANT)
- Premium luxury UI (inspired by high-end brands)
- Color palette:
  - Primary: Black (#0F0F0F)

  - Accent: White / Cream
- Smooth animations using Framer Motion
